# Book Base サムネイル作成ルール

## 目的

Book BaseのサムネイルはYouTube動画のクリック率を高めるための画像である。動画内画像とは役割が異なり、最優先は視聴者が思わずクリックしたくなること。ただし単なる煽り画像にはせず、水彩画風、高級感、知的で落ち着いた雰囲気、ビジネス書・教養書チャンネルらしさ、会社員が自分に関係あると感じる切り口を守る。

## 基本仕様

- アスペクト比は必ず16:9。
- 今日の日付の`YYYYMMDD_book_cover`画像を使用する。
- 視聴者がクリックしたくなる独自コメントを大きく表示する。
- ABテスト用に3パターン作成する。
- Pattern Aは脅し・損失回避型、Pattern Bは誘惑・ベネフィット型、Pattern Cは逆張り・好奇心型にする。
- 各パターンでコメントの切り口を明確に変える。同じ内容の言い換えだけにしない。
- イラストは水彩画風の高級感があるものにする。
- Book Baseロゴを小さく自然に入れる。
- 文字は少なめにし、一目で意味が分かるようにする。

## ブックカバー使用ルール

- ブックカバーは必ずサムネイル内に入れる。
- 文字が読める程度に十分大きく表示する。
- コメントより主役になりすぎないようにバランスを取る。
- ブックカバーを歪ませない。
- ブックカバーを雑に切り抜かない。
- 背景になじませつつ、視認性は確保する。
- ブックカバーがない場合はサムネイル生成を`needs_review`にし、架空の表紙や無関係な本画像を作らない。

## 3パターン

### Pattern A：脅し・損失回避型

- 目的：知らないと損をする、このままだと危ないと感じさせる。
- 狙う感情：不安、焦り、損失回避、置いていかれたくない気持ち、危機感。
- コメント例：「その努力、逆効果です」「頑張るほど遠回り」。
- 避けること：下品な煽り、誇張、何が損なのか分からない曖昧表現、他パターンと同じ意味。

### Pattern B：誘惑・ベネフィット型

- 目的：これを知れば得できる、自分もこうなりたいと思わせる。
- 狙う感情：期待、成長欲、得したい気持ち、年収アップや仕事改善への希望。
- コメント例：「仕事がラクになる考え方」「一流はここが違う」。
- 避けること：誰でも成功系の安っぽい表現、弱いベネフィット、本の内容とズレる表現。

### Pattern C：逆張り・好奇心型

- 目的：え、そうなの？普通はそう思わない、と感じさせる。
- 狙う感情：意外性、違和感、好奇心、常識がひっくり返る面白さ。
- コメント例：「努力しない方が続きます」「頑張る前に捨てるべきです」。
- 避けること：釣りタイトル、本の主張を歪める過激な逆張り、A/Bと違いが弱いコピー。

## コメント作成ルール

- 一瞬で理解できる短さにする。
- できるだけ1メッセージに絞る。
- 原稿の文章をそのまま使わない。
- 書籍の内容を、視聴者がクリックしたくなる言葉に変換する。
- その本ならではの主張が感じられる言葉にする。
- 動画タイトルの単なる短縮版にしない。
- 3パターンで切り口を変える。
- 目安は7〜18文字、長くても25文字以内。2行構成までは可、3行以上は避ける。
- 「知らないと損」「保存版」「誰でもできる」「必見」「会社員必見」「今すぐ見て」「これだけでOK」「人生が変わる」は単独で使わない。

## デザインルール

- 全体トーン：水彩画風、高級感、知的、落ち着き、洗練、ビジネス書要約チャンネルらしさ。
- ベースカラー：淡いクリーム、白、ベージュ、ライトグレー。
- アクセントカラー：ティール、青緑、ゴールド、ネイビー、ブラウン。
- 避ける色：原色の多用、蛍光色、真っ黒背景、派手なグラデーション、安っぽい赤黄の強調。
- 独自コメントを主役にし、ブックカバーも見やすく配置する。
- 余白を取り、スマホでも読める大きさにする。
- 3パターンとも同じ構図にせず、コメント位置、ブックカバー位置、背景の見せ方を少しずつ変える。
- Book Baseロゴは右上、右下、左下のいずれかに小さく自然に配置する。

## 出力ファイル名

- `thumbnail_A_loss_aversion.png`
- `thumbnail_B_benefit.png`
- `thumbnail_C_curiosity.png`
- 使用コメントは`thumbnail_comments.md`に保存する。

## 品質チェック項目

- 16:9になっているか。
- ブックカバーが使用されているか。
- 独自コメントが大きく表示されているか。
- Pattern Aが損失回避型、Pattern Bがベネフィット型、Pattern Cが逆張り・好奇心型になっているか。
- 3パターンの切り口が十分に違うか。
- 本の内容とズレた煽りになっていないか。
- 水彩画風・高級感・Book Baseらしい知的な雰囲気があるか。
- スマホで読める文字量か。
- ブックカバーの視認性があるか。
- 汎用的な弱いコピーになっていないか。
