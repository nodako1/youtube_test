# 画像プロンプト作成ルール

- 20シーン分の画像生成プロンプトを作る。
- 画像プロンプト生成時は、`script_rules.md`、`image_rules_common.md`、`image_block_rules.md`、該当する`image_scene_rules/scene_XX.md`、現在のシーン原稿、前後シーンの画像メタ情報の順で参照する。
- 各シーンごとに、シーン番号、所属ブロック、ブロックの役割、重要ポイント番号、ブロック内での役割、前ブロックからの理解の流れ、このシーンで伝える要点、画像の目的、推奨構図、画面内テキスト、前後画像との差別化、最終プロンプトを出力する。
- シーン5〜16では、重要ポイント1・2・3のどれに該当するか、動画全体の理解の流れでどんな役割を持つか、前の重要ポイントからどう深まり、次の重要ポイントへどうつながるかを必ず明記する。
- 同じ構図が連続しないようにする。
- 原稿本文をそのまま長文で載せない。
- 1画像1メッセージにする。
- 水彩画風で、高級感のある落ち着いたビジネス書チャンネル向けにする。
- 色味は淡いクリーム、白、ベージュ、ティール、ゴールドを中心にする。
- Book Baseロゴを右下または右上に小さく自然に配置する。
- v1では画像ファイル生成ではなく、画像プロンプト作成までを対象にする。
