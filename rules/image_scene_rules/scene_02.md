# scene_02

## 固定役割

scene_02は、scene_01のクイズの正解を発表し、その正解の意味づけ、世間のイメージとのズレ、本書が示す本質、この後に学ぶテーマまでを一画面で自然につなげるシーンである。

## 恒久ルール

- 正解発表シーンにする。
- scene_01の答えを一目で見せる。
- 正解だけで終わらせず、原稿後半の意味づけ情報も見せる。
- 画面の約50%を正解発表エリア、残り約50%を補足情報エリアにする。
- 補足情報エリアには「世間のイメージとのズレ」「本書が示す本質」「この後に学ぶテーマ」を短く表示する。
- 左側または中央に「正解：〜」を大きく表示し、反対側に「一般的な見方 → 本書の視点」のような対比情報を置く。
- 今回のテーマへ橋渡しする。
- 派手なクイズ番組風にしない。
- 短いテキストで見せる。ただし文字が少なすぎて内容が伝わらない画像にはしない。
- 特定の本・著者・テーマに依存した言葉を固定しない。

## 可変データ

毎回、scene_01とscene_02の原稿、および `image_context.json` から以下を抽出する。

```json
{
  "scene": 2,
  "fixed_role": "クイズ正解発表と意味づけ・テーマへの橋渡し",
  "quiz_correct_answer": "A / B / C のいずれか",
  "answer_short_label": "正解の内容を短く表す言葉",
  "public_gap_label": "世間のイメージとのズレを短く表す言葉",
  "book_essence_label": "本書が示す本質を短く表す言葉",
  "theme_bridge_label": "この後に学ぶテーマに接続する短い言葉",
  "exact_text_elements": ["正解は〇", "答えの短い内容", "一般的な見方→本書の視点", "学ぶテーマ：〇〇"]
}
```

## 禁止事項

- 正解だけを大きく表示して終わる画像。
- A/B/Cの答えだけで情報が完結する画像。
- 原稿後半の意味づけと無関係なイラスト。
- 文字が少なすぎて内容が伝わらない画像。
- AIにBook Baseロゴを描かせること。固定ロゴ画像を後処理で合成する前提にする。

- 特定動画の語句が恒久テンプレートに固定されていないか。
