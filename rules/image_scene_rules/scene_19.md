# scene_19

## 固定役割

現在の本と関連過去動画の本を自然につなぎ、学びの連続性・理解の広がり・次の学びへの導線を見せる接続シーン。広告ではなく、視聴者の理解を深めるための接続シーンとして機能させる。

## 画像の目的

今回の本で得た学びが、過去動画で紹介した別の本の内容ともつながっていることを、視聴者が自然に理解できるように見せる。

## 毎回生成する可変データ

scene_19では、現在の原稿・設定・関連過去動画情報から以下を毎回抽出・定義する。

- current_book_label：今回の本のテーマを短く表す言葉
- related_book_label：過去動画の本のテーマを短く表す言葉
- connection_reason：2冊の学びがどのようにつながるかを短く説明する言葉
- connection_message：視聴者に伝える短い接続メッセージ
- connection_type：same_theme / next_step / practical_extension / contrast_and_deepen / money_or_life_application のいずれか
- visual_structure：book_bridge / light_path / flowing_pages / ribbon_connection / two_books_perspective のいずれか
- exact_text_elements：画像内に表示する短い日本語テキスト。最大1要素

## 固定してよい表現

- 学びがつながる
- 理解が深まる
- 次の一冊へ
- 現在の本と過去動画の本
- 2冊の本の接続表現
- 橋、光の道、リボン、流れるページ、やわらかな線
- Book Baseらしい上品で落ち着いた接続シーン
- 宣伝ではなく、学びの連続性を伝える構図

## 固定してはいけない表現

- 今回の本の具体的なタイトル
- 今回の本の具体的なキーワード
- 特定の著者名
- 過去動画の本の具体的なタイトル
- 過去動画の具体的テーマ
- 現在の原稿にしか出てこない言葉
- 特定ジャンルに寄りすぎるモチーフ

これらは固定テンプレートへ書かず、毎回の原稿・設定・関連動画情報から可変内容として生成する。

## 文字ルール

- 画像内テキストは最大1要素まで
- 短い日本語のみ
- 12文字前後を目安にし、長くても15文字程度まで
- 説明文にしない
- 本のタイトルや著者名を無理に入れない
- 原稿本文のコピペ、CTA文、英語テキスト、長文、意味不明な文字は禁止

使用してよい固定見出し候補は「学びがつながる」「理解が深まる」「次の一冊へ」のうち1つだけ。

## 構図ルール

- 2冊の本を必ず出す
- 参照画像は現在の本の表紙として扱う
- 過去動画側の本を省略しない
- 2冊をただ横に並べず、橋・光・ページ・線などで自然につなぐ
- scene_20の締め画像と混同しない
- 広告バナー風・CTA風・クリック誘導風にしない

## 推奨構図

- 光の橋型：左側に今回の本、右側に過去動画の本。2冊の間を柔らかく光る橋でつなぐ
- ページの流れ型：今回の本のページから光や紙片が流れ、別の本へつながる
- 奥行きのある2冊構図：手前に今回の本、奥に関連する過去動画の本。淡い線や光で関係性を見せる
- リボン接続型：2冊の本を細いリボンや線で控えめにつなぐ

## 避けること

関連動画を見てください、こちらもおすすめ、クリック、今すぐ見る、強い矢印、派手な赤や黄色、サムネイル風の煽り、参照画像だけを大きく表示する、表紙広告のように見せる、scene_20の締め画像のような終了感。
