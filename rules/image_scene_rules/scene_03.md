# scene_03

## 固定役割

scene_03は、今回紹介する本を提示し、動画冒頭の結論と補足要点を1枚で伝えるシーンである。

## 恒久ルール

- 現在のブックカバーを左側に大きく、自然な主役として表示する。
- ブックカバーは後処理で貼り付けず、画像生成開始時点で実際のbook_cover画像を参照画像として渡す。
- 実際のブックカバー画像を前提に、全体の背景・光・余白・右側情報を構成する。
- AIにブックカバーを別物として再生成させない。タイトルだけ似せた架空本も禁止する。
- 右側には「今回の1冊」ラベル、動画内の結論、補足要点2〜3個、対象者・メリットを簡潔に整理する。
- 背景はBook Baseらしい水彩・淡いクリーム・ティール・ゴールドの高級感あるビジネス書要約チャンネル風にする。
- 背景イラストは本のテーマとリンクする人物、仕事、会話、学習、相談、成長などから選ぶ。
- book_coverパスは固定せず、毎回 `input/YYYYMMDD_book_cover.*` から動的に取得する。

## 可変データ

毎回、input内の現在日付ファイルと原稿から以下を抽出する。

```json
{
  "scene": 3,
  "fixed_role": "今回の本紹介と動画全体の結論提示",
  "current_book_cover_path": "現在の本のbook_cover画像パス",
  "book_title": "現在の本のタイトル",
  "author_name": "現在の著者名",
  "video_conclusion": "動画全体の結論を短く表す言葉",
  "supporting_points": ["補足要点1", "補足要点2", "補足要点3"],
  "target_viewer_label": "〜したい人におすすめ",
  "exact_text_elements": ["今回の1冊", "動画の結論", "補足要点1", "補足要点2", "補足要点3", "〜したい人におすすめ"]
}
```

## プロンプト必須文

Use only the following Japanese text elements exactly as written. Do not add any other Japanese or English text.

## 構図テンプレート

- 左側：参照画像として渡した実ブックカバーを大きく配置し、背景に自然に馴染ませる。
- 右上：「今回の1冊」ラベル。
- 右中央：動画の結論を最も大きく表示。
- 右中段〜下段：補足要点を2〜3個、アイコン付きまたは小見出し付きカードで表示。
- 右下または背景：内容とリンクした人物イラスト、仕事シーン、会話シーン、学習シーンなど。

## 最終チェック

- 実ブックカバーを後貼りせず、参照画像として生成に渡しているか。
- 表紙が背景と光・影・余白の面で自然に馴染んでいるか。
- 左に書影、右に結論・要点・訴求が整理されているか。
- 右側が意味の薄い飾りだけになっていないか。
- 固定パスや特定タイトルをテンプレートに入れていないか。
- 結論と補足要点は今回の原稿から生成されているか。
