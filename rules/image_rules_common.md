# Book Base 動画内画像 共通ルール

- 画像作成時は各シーン本文だけでなく、動画全体の20シーン構造と所属ブロックを必ず確認する。
- 20シーンは5ブロックで捉える：シーン1〜4は導入・結論・全体像、5〜8は重要ポイント1、9〜12は重要ポイント2、13〜16は重要ポイント3、17〜20はおさらい・実践・関連動画接続・締め。
- 重要ポイント1・2・3は単なる並列ではなく、土台・入口から、理由・構造の理解深化へ進み、最後に実践・応用・行動化へ落とし込む流れとして扱う。
- 原稿本文をそのまま画像内に長文で載せず、1画像1メッセージに絞る。文字なしが適切な場合は文字なしでもよい。
- 画像内テキストを入れる場合は短いキーワードまたは短い見出しにする。
- イラストは水彩画風で、高級感のある落ち着いたビジネス書チャンネル向けにする。
- 色味は淡いクリーム、白、ベージュ、ティール、ゴールドを中心にする。
- 構図は前後画像と被らないように、人物位置、視点、背景、モチーフ、文字配置を変える。
- Book Baseロゴを右下または右上に小さく自然に配置する。
- 見た目の綺麗さだけでなく、必ずシーンの役割、所属ブロック、重要ポイント1・2・3の流れに合った画像にする。
